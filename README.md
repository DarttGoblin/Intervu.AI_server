# 🎥 Intervu.ai – Backend  

This is the **backend** of Intervu.ai, an AI-powered mock interview assistant.  
It powers the application’s **AI-driven features**, including generating questions, analyzing responses, and handling audio processing. 🚀  

---

## ✨ Features  

- 🧠 **AI Question Generator** – Provides interview questions dynamically.  
- 🔊 **Text-to-Speech (TTS)** – Converts questions into natural audio using gTTS.  
- 🎤 **Speech-to-Text (STT)** – Uses Whisper to transcribe interview answers.  
- 🔗 **API Endpoints** – Connects seamlessly with the frontend.  

---

## 🛠️ Tech Stack  

- **Backend**: Flask (Python)  
- **Speech Processing**: gTTS (Text-to-Speech), Whisper (Speech-to-Text)  
- **AI Integration**: Custom interview question generation
